1.Difference between flex row and column

CSS property based on concept of flex parent element and child element

flex-direction:row

If the element is placed in row that x-axis is known as main axis and y-axis would be called as cross axis

flex-direction:column

In flex direction column all child element would be coming in vertical manner,child placed one after another in column manner and y-axis called as main axis, x-axis would be called as cross axis

2.Explain inline , internal and external stylesheet

Inline-Gets applied to the element in the style attributes.Style applied to HTML individually, style is having the highest priority among other two and not advisable to use

< p style="color : red" > text < /p>

Internal-Style is defined in the < head > of HTML file

          < head >
               < style >
                      p { color: red}
               < /style>
          < /head>

External- By creating a different css file (index.css) attached with link tag to the head tag of HTML

        p{
            background - color :red
         }
